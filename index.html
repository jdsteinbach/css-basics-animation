Little Known CSS Basics

The Box Model

Everything Is A Rectangle

All rectangles have:

  • height
  • width

Height defaults to content height

Width defaults to block formatting default

  • 100% for block elements
  • Content-width for inline / inline-block elements

Override width and height with:

  • explicit width & height properties
  • also min- and max- versions

What Affects “Default” Width & Height?

CSS!

  • margin
  • border
  • padding
  • contents

What Controls which Properties Affect “Default” Width & Height?

box-sizing

  • content-box: padding affects element size
  • border-box: padding doesn't affect element size
