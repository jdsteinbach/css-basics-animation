CSS Color Formats

Named Colors

RGB

Red, green, & blue values (0-255)

rgb(96,86,156) rgb(92,100,67) rgb(218,201,53)

RGBA

RGB (0-255), alpha transparency (0-1)

rgb(96,86,156,.3) rgb(92,100,67,.5) rgb(218,201,53,.8)

Hexadecimal

Also red, green, blue values,
but ranges from 00–ff

3 / 6 digit: #000 = #000000

#60569c #5c6443 #dac935 #ebecf0 #2c313e

Hexadecimal+

Future CSS: alpha transparency
4 / 8 digit:#000a = #000000aa

#60569c4c #5c64437f #dac935cc

HSL

Hue (0-360), saturation (0-100%), & lightness (0-100%)

hsl(249,29%,47%) hsl(75,20%,33%) hsl(54,69%,53%)

HSLA

HSL + alpha transparency (0-1)

rgb(96,86,156,.3) rgb(92,100,67,.5) rgb(218,201,53,.8)

currentColor

The OG CSS variable

currentColor = the CSS color applied to the selected element
